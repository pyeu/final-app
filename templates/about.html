About Deaths in Custody

Data

My data comes from the Deaths in Custody Reporting Program , a program within the Bureau of Justice that collects data on deaths that occur during the arrest process or while inmates are in the custody of local jails or state prisons. However, the BJS does not publish their findings to the public.

I found the data through a MuckRock request filed by Dave Maass with the Department of Justice in California, thanks to Dan Nguyen's explainer on how to filter by completed MuckRock requests.

As far as data manipulation, the keys to the codes for 'custody_offense' and 'agency_name' came from seperate spreadsheets in the same MuckRequest, which I joined to the main dataset. I added variables for "birthdate" and "age" so that users could sort by youngest or oldest inmate.

Limitations

Since this app is based off a one-time FOIA request, the data has not updated since it was requested in April 2015. Data is for 2013 only.

I'd originally hoped to run analysis on death rates by days in custody, but the Bureau of Justice either doesn't keep track of days in custody or didn't release that data along with this particular request. Plus, there's no easy way to correlate inmate names with arrest records, so I ended up letting that one go.

Technical Stack

Data cleaning and analysis was done in Python, with help from the Pandas library. The web app is powered by Flask. The charts are generated using Highcharts, with help, as always, from Dan.
